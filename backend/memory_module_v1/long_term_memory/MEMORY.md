# Long-term Memory

## 系统背景

ClawTeam 是一个医疗多智能体协作诊疗系统。本文件用于存储长期医学知识、常见诊疗路径、以及用户会话中沉淀的重要上下文。

## 常用诊疗路径（示例）

### 上呼吸道感染
- 主诉特征：咳嗽、咽痛、鼻塞、低热
- 病程提示：病毒性多在 7-10 天内自愈；细菌性需考虑抗生素
- 危险信号：高热持续不退、呼吸困难、胸痛、咳血 → 建议立即就医

### 胸部影像检查选择
- 筛查首选：胸部 X 线（辐射低、成本低）
- 发现异常进一步：低剂量胸部 CT（结节、肿块评估）
- 软组织/纵隔病变：增强 CT 或 MRI

### 常见药物相互作用提示
- 抗生素 + 抗凝药（华法林）：可能增强抗凝效果，需监测 INR
- NSAIDs（布洛芬）+ 抗高血压药：可能降低降压效果
- 抗生素 + 口服避孕药：部分抗生素可能降低避孕效果
